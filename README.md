# Decentralized Persistent Monitoring with Deep RL on Networks
_Theodore Rogalski_ 

This repository contains the tools necesessary for training, testing and customizing teams of decentralized agents for the persistent monitoring problem on networks. 

## Features
* Custom network environment creation for RL training (stored in .graphml files)
* Fully decentralized teams of an arbitrary number of agents that learn using deep reinforcement learning

## Installation
The installation of the dpmrl system onto your computer involves the following four high level steps:
1. Use the following command to git clone the repo:
``` bash
git clone https://github.com/theohrogalski/dpmrl
```
2. Once inside the parent directory _(dpmrl)_ the following command to install the required packages:
``` bash
pip install -r requirements.txt
```
3. Use the following command to go into the custom_environment folder:
``` bash
cd custom_environment 
```
4. Then, use the 
Install guide:
- git clone this repo
- install requirements.txt into a venv
